## 공통 촬영 규칙

### 화면 구성
- 왼쪽 70%: Claude Desktop 메인 채팅창
- 오른쪽 30%: 툴 호출 패널 (`⚙️` 아이콘 펼친 상태)
- 폰트: 노출 가독성을 위해 Claude Desktop 설정 → **Text size: Large**
- 컬러 테마: **Light**로 통일 (어두운 테마에서 코드 블록 대비가 떨어짐)

### 녹화 사양
- 해상도: 1440×900, 녹화 후 720px 너비로 리사이즈
- 프레임레이트: 24fps (GIF 용량 최적화)
- 최종 포맷: `.gif`, 최대 8MB 이하 (GitHub 인라인 표시 제약)
- 오디오: 없음. 모든 설명은 자막 오버레이.

### 자막 스타일
- 위치: 하단 중앙, 반투명 배경 (#000 @ 70%)
- 폰트: Pretendard Bold 28px
- 색상: 흰색, 강조 어구는 `#FFD93D` (핵심 숫자·부처명)
- 지속시간: 한 자막당 최소 2초, 읽는 데 필요한 시간 확보

### 커서 동작
- 타이핑은 **실시간 타이핑 그대로** 녹화 금지. 사전 입력 후 재생 속도 1.5x로 가속.
- 결과 스크롤은 **0.5x로 감속**하여 Claude가 생성하는 토큰을 눈으로 따라갈 수 있게 한다.
