# MCP Question Examples

이 문서는 Govpress MCP를 실제로 써볼 때 바로 붙여 넣을 수 있는 내용 중심 질문 예시 모음이다.

기술 검증용이 아니라, 정책 흐름·부처 비교·최근 발표 탐색 같은 실제 활용 시나리오에 맞춘 예시만 모았다.

## 정책 흐름 추적

- 최근 3년간 `탄소중립` 정책 흐름을 시계열로 정리해줘.
- `디지털플랫폼정부` 정책이 2022년부터 2026년까지 어떻게 바뀌었는지 설명해줘.
- `우주항공` 관련 정책 발표 흐름을 시간순으로 보여주고, 전환점이 된 발표를 짚어줘.
- `기후위기 적응` 정책이 최근 몇 년간 어떤 방향으로 구체화됐는지 정리해줘.
- `저출생` 대응 정책에서 최근 1년간 발표가 가장 많은 부처와 핵심 메시지를 정리해줘.

## 부처 간 비교

- `AI 인재양성`에 대해 부처별로 목표 숫자나 표현 차이가 있는지 비교해줘.
- `반도체 인재` 관련해서 교육부, 산업부, 과기정통부 입장을 비교해줘.
- `청년 주거` 정책에서 국토부와 다른 부처 발표가 어떻게 연결되는지 찾아줘.
- `탄소중립` 관련 목표연도나 핵심 수치가 서로 다르게 발표된 사례가 있는지 비교해줘.
- `AI 인재 50만` 같은 숫자 목표가 부처별로 다르게 쓰인 사례가 있는지 찾아줘.

## 최신 발표 탐색

- `기후위기 적응` 관련 최신 보도자료 5건을 핵심만 요약해줘.
- `전기차 화재` 또는 `배터리 안전` 관련 정부 대응 발표를 모아서 비교해줘.
- `의대 정원` 관련 최신 발표들을 모아 핵심 입장만 정리해줘.
- `공급망` 관련 최근 발표 중 정책 변화가 보이는 문서만 골라줘.
- `첨단산업` 관련 이번 달 발표를 부처별로 나눠서 요약해줘.

## 변화·불일치 탐색

- `의대 정원` 관련 보도자료 흐름을 정리하고, 시기별로 톤이나 초점이 어떻게 달라졌는지 분석해줘.
- `탄소중립` 정책에서 최근 발표와 초기 발표를 비교해 핵심 표현이 어떻게 달라졌는지 보여줘.
- `AI 인재양성` 관련 발표 중 숫자, 목표연도, 주관부처가 충돌하는 사례가 있는지 찾아줘.
- `반도체 특별법` 관련 발표에서 부처별로 강조점이 어떻게 다른지 정리해줘.
- `디지털플랫폼정부` 정책에서 2024년 이후 강조점이 이전과 어떻게 달라졌는지 설명해줘.

## 추천 사용 패턴

질문을 더 잘 먹이려면 아래 식으로 쓰면 좋다.

- 주제 + 기간:
  - `2023년 이후 탄소중립 정책 흐름을 정리해줘`
- 주제 + 비교축:
  - `AI 인재양성에 대한 부처별 입장 차이를 비교해줘`
- 주제 + 목적:
  - `기후위기 적응 관련 최신 발표를 기자 브리핑용으로 요약해줘`
